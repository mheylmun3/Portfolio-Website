American Housing Market Analysis

This project was the Final Project for my Statistical Computing and Graphics with R class. In this project, I analyzed U.S. Census data to study trends in housing and rent prices, identifying which demographics were most impacted by changes using R. This project was meant to be performed in groups of three, but I decided to complete this by myself. Despite this, my project was recognized as the top project in both class sections by the professor.
